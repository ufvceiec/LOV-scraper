# LOV scraper

**Summary**: \
Python script to download all vocabs and some information available from Linked Open Vocabulary (LOV) project (https://lov.linkeddata.es/dataset/lov/vocabs). The oupout is a csv file.

**Requirements**: 
- selenium
- csv
- time
- firefox web browser
