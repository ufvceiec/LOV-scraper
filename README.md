# Linkeddata-Scraper

**Summary**: \
Python script to download all vocabs and some information available in linkeddata web (https://lov.linkeddata.es/dataset/lov/vocabs) in tabular format and in a CSV file, using selenium.

**Requirements**: 
- selenium
- csv
- time
- firefox web browser
